# DJIFLA
DJI Flight Log Analyzer

## DJI Fly のフライトログデータを解析する PWA アプリです。
DJI製ドローンの飛行データログを解析して、離陸時刻、着陸時刻、飛行時間（分）を取り出します。

このアプリは、iPhone で動作を確認しています。

Android端末での動作は未確認です。

## PWA アプリなので、スマホにインストールして使います。
このファイルをそのままで使いたい人は、公開しているサイトを開いて、ホーム画面にインストールして利用できます。
公開URL: http://ynomura.com/DJIFLA/

修正するなどして使いたい方は、「wpa」フォルダごとダウンロードし、中のファイルを修正して、フォルダ名は好みの名称として、自サイトにアップロードして使います。

## 操作手順
iPhone の DJI Fly アプリで作成されたログデータは拡張子が .txt ですが、中身はバイナリデータなので、そのままでは使えないので、DJI  Flight Log Viewer のオンラインサービスを使います。
https://www.phantomhelp.com/logviewer/upload/

フライトログファイルは、「この iPhone 内」→「DJI Fly」→「FlightRecords」の中に、「FlightRecord_YYYY-MM-DD_[HH=MM-SS].txt」のようなファイル名で登録されているので、それを指定してアップロードして変換してもらいます。

変換が終了すると、Download CSV をタップしてダウンロードできます。

ダウンロードしたファイルは、zip 圧縮されていますが、そのままこのアプリで変換・解凍して csv ファイルとして解析します。

複数の ZIP ファイルを連続して解析して表の下に追加します。（表のデータをクリアしたいときは「クリア」ボタンを押します）

解析が終了したら、「TSVをコピー」ボタンを押して結果をクリップボードにコピーします。

スプレッドシートに貼り付けます。

## AppleScript 版を追加しました
同じように csv に変換済みのファイルを解析して、日誌に必要なデータを取り出す機能を AppleScript で書いたものを追加しました。



他にもファイルがある場合は、「クリア」ボタンを押して繰り返します。


解凍した csv ファイルを、このアプリにアップロードすると、変換した結果を TSV（タブ区切りのテキスト）としてクリップボードにコピーされるので、それをスプレッドシートにペーストして使います。
